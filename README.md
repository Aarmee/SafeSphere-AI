# SafeSphere AI

Predictive Safety Intelligence Platform

## Tech Stack

- React
- Spring Boot
- PostgreSQL
- FastAPI
- Docker
- Kafka
- Redis

## Problem Statement

Most safety applications focus on emergency response after an incident occurs.

SafeSphere AI aims to shift from reactive safety to predictive safety through AI-powered risk analysis, safe route recommendations, community incident intelligence, and offline emergency communication.

## Core Features

- SOS Emergency System
- Guardian Management
- Live Location Tracking
- AI Risk Prediction
- Safe Route Recommendation
- Community Incident Reporting
- Safety Heatmaps
- Check-In Timer
- Offline SMS Fallback
